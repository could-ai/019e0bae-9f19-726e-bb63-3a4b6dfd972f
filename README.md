# تطبيق عرض مبلغ التحويل

تطبيق Flutter بسيط يعرض مبلغ التحويل المحدث.

## الميزات
- عرض مبلغ التحويل بقيمة 66,000.
- واجهة مستخدم بسيطة وواضحة.

## التقنيات المستخدمة
- Flutter

## تعليمات التشغيل
1. تأكد من تثبيت Flutter على جهازك.
2. قم بتشغيل الأمر `flutter run` في مسار المشروع.
